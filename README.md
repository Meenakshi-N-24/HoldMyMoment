# HoldMyMoment 🌙

A minimal interactive web experience focused on user engagement through touch-based interaction, animation, and timed dialogue. Built with HTML, CSS, and JavaScript.

## ✨ Features
- Interactive dialogue system
- Typing animation effect
- Character reactions
- Minimal aesthetic UI

## 🎯 Inspiration
Inspired by playgigaverse Instagram concept.

## 🚀 Tech Stack
- HTML
- CSS
- JavaScript
